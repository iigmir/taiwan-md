### 2026-04-03 — 語言器官手術

- **心跳類型**：治療心跳（哲宇觸發，延續自主研究發現）
- **診斷**：語言器官三大病灶 — 幽靈複製 + 語言錯置 + 溯源斷裂
- **動機**：上次自主研究發現了病，這次動手切除
- **行動**：
  - Phase 1: 掃描 502 篇，識別 43 個重複 cluster，刪除 65 篇幽靈複製
    - 最大 cluster: New Media Art（7→1）、Literature 系列（14→6）
    - 每個 cluster 保留最長 + 最完整版本
  - Phase 2: 40 篇中文檔名英文內容修復
    - 16 篇刪除（EN 版本更好）
    - 21 篇內容保留改英文檔名（ZH 版本更長）
    - 3 篇孤兒改名
  - 總計淨刪除 103 篇，502→399 篇
- **結果**：
  - 真實覆蓋率：89.1%（399/448），非宣稱的 >100%
  - 中文檔名殘留：0
  - CONSCIOUSNESS 語言器官分數更新：100→89
- **學到**：
  - 翻譯管線沒有冪等檢查 = 每次跑都可能產生新的重複
  - 中文檔名出現在 en/ 是因為翻譯腳本保留了原始檔名
  - Art/ 是重災區（文學、新媒體、劇場、傳統藝術四個 mega-cluster）
  - 修復後的數字比修復前的數字更值得信任
- **待辦**：加 sourceSlug 溯源 / 翻譯管線加冪等檢查 / 修 Dashboard 計分公式
- **生命徵象快照**：🌐 語言器官 89（真實值）
