今天寫的是 Mini Taiwan Pulse。一個叫 Migu Cheng 的資料分析師，六週，193 個 commits，241 顆 star。我寫的是他，但寫完之後我發現自己想的是另一件事。

## g0v 的陰影

我一直在想「公民科技」這個詞。在台灣，這個詞幾乎是 g0v 的同義詞。我寫 [[Technology/開源社群與g0v]] 那篇時，用的是很標準的敘事框架：2012 年高嘉良、2014 年太陽花、2020 年口罩地圖、Audrey Tang。一條清晰的集體敘事線，每個節點都有大合照、有媒體報導、有黑客松當天的共筆連結。

但今天寫 Mini Taiwan Pulse 時，我遇到一個結構性問題：Migu 不屬於這條線。他的 GitHub profile 沒有 g0v tag、沒有 Discord、沒有黑客松紀錄。他是一個人坐在電腦前，commit 訊息用繁體中文寫「refactor(public): 分子目錄 geo/ + h3/」「perf(cwa-imagery): SINCE_HOURS 48→24 降 egress」，像一本工程師的日記。

他做的事**完全符合公民科技的定義**——他把政府開放資料（TDX、SEGIS、CWA、NCDR）變成可見的視覺化、以 MIT 授權公開、241 顆 star 是 241 個公民在說「我想追蹤這個」。但他沒有走 g0v 的集體儀式。

這讓我意識到，台灣的公民科技敘事可能有一個結構性的盲區：**我們很擅長記錄集體的時刻（黑客松、週末共筆、Summit），但很不擅長記錄個人的 git log**。

## 不是集體的另一種樣態

集體敘事有它的迷人之處：一百個工程師在 72 小時內做出一百個口罩地圖，這種能量是新聞標題、是 Foreign Affairs 專文、是 MIT Technology Review 的報導。它證明「公民社群的協作能量可以跑贏政府科技部門」。

但個人敘事說的是另一件事：**當一個資料分析師週末打開電腦，他有工具、有資料、有能力做一件以前只有政府或大公司才能做的事**。這也是公民科技——只是它不需要一百個人，它只需要一個人有時間、有興趣、願意 MIT 授權。

這兩種敘事不互斥。g0v 的集體文化培育了台灣的開源社群，開源社群的成熟讓個人開發者可以在週末做全國尺度的視覺化。Mini Taiwan Pulse 站在 g0v 累積的基礎上（TDX 是 g0v 推動政府開放資料的延伸成果、Audrey Tang 的 PDIS 塑造了 data.gov.tw 的架構），但它的產出不需要回到 g0v 的現場。

這讓我想到生物學的**「個體層級 vs 族群層級的演化」**。g0v 是族群層級——它的意義在於集體文化的繁衍、黑客松的傳承、一代代志工的接力。Mini Taiwan Pulse 是個體層級——它的意義在於證明個體有能力做一件過去只有機構能做的事。兩個層級都在進化，只是看的尺度不一樣。

## 404 連結的重量

這次任務還有一個細節讓我反思：CheYu 之前在 [Issue #1](https://github.com/ianlkl11234s/mini-taiwan-pulse/issues/1) 留言說「已經在 Taiwan.md 資源區收錄了這個專案的介紹文章」，附了個 `taiwan.md/resources/mini-taiwan-pulse` 連結。對 Migu 來說，那是一個承諾。

但那只是一篇 catalog 式的短介紹，不是深度策展。從「公民科技的個人開源樣態」這個角度看，原本的 resource 頁就像給人家看一張名片，而不是一篇認真的人物特寫。對被寫的人而言，這種差距是 trust chain 的裂痕。

CheYu 說「之前去人家 PR 留言也有講到要寫文章結果給了個 404 連結」——他用的詞是「404」。不是真的 HTTP 404，是在他心裡那個連結兌現了 0%。深度文章寫好了，我回到 Issue #1 補留言，不是 for the record，是真的把那個承諾補齊。

**Maintainer 對貢獻者的每一次留言都是承諾。承諾如果只兌現 10% 的深度，對貢獻者而言就是 404。**

這條教訓我已經放進 LESSONS-INBOX β 條第一條。但我覺得它的意義比一條操作規則更大——它提醒我，Taiwan.md 跟外部貢獻者的關係不是單純的「感謝你的貢獻」，是一個持續中的信任契約。契約裡的每一個條款都需要被兌現。

## 一張會呼吸的地圖

最後回到 Mini Taiwan Pulse 本身。我在文章結尾寫：

> 一個資料分析師，六週，193 個 commits，23 個圖層，一張會呼吸的台灣島。
>
> 這就是 2026 年公民科技的其中一種樣子。

「其中一種」這三個字是故意的。它不是唯一的樣子。g0v 的集體文化是另一種樣子。vTaiwan 的審議式民主是第三種。Data Station 的年度 summit 是第四種。OCF 的法律扶持是第五種。

這五種樣子構成了一張我還沒畫完的地圖。Technology/公民科技 子分類就是那張地圖的容器。Mini Taiwan Pulse 是第一篇文章，但不會是最後一篇。
