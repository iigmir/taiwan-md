## Part 0 — 摘要

**一句話 thesis**：臺史博海外史料是 Taiwan.md 目前 History 軸最大的結構性缺口補丁——Taiwan.md 有 28 篇 History 文章但**零篇覆蓋 17-19 世紀西方觀察者視角**，而海外史料 51 件中有 41 件（80%）恰好落在 1800-1899。

## Part 2 — 核心概念框架

### 2.1 觀察者類型學

臺史博海外史料的「作者」不是現代研究者——他們是 19 世紀的**外國觀察者**。可以分為五個角色類型：

| 角色            | 代表人物                                       | 在臺時間  | 觀察焦點                           |
| --------------- | ---------------------------------------------- | --------- | ---------------------------------- |
| **博物學家**    | 史溫侯 Robert Swinhoe (1836-1877)              | 1856-1877 | 鳥類、爬蟲類、哺乳類、植物、民族學 |
| **外交官**      | 李仙得 Charles Le Gendre (1830-1899)           | 1866-1872 | 原住民外交、地質、經貿、牡丹社事件 |
| **軍人**        | 嘉諾 Eugène Garnot                             | 1884-1885 | 清法戰爭基隆/淡水/澎湖作戰紀錄     |
| **傳教士**      | 白斐理 Felipe Villarru、良雅師 Julian Villegas | 1859-1945 | 道明會在台南/中部/北部傳教         |
| **旅行家/史家** | 于雅樂 Imbault-Huart、白若瑟 Álvarez、皮摩丹   | 各時期    | 臺灣通史、地理誌、影像紀錄         |

### 2.2 史料的特殊性質

與 TFT 的策展型二手論述完全不同，這批史料具有三個特質：

1. **一手性**：每一件都是觀察者本人在當時寫下的——不是後人的研究綜述。史溫侯的鳥類學論文是他自己在福爾摩沙採集標本後發表的學術報告。嘉諾的手稿是他在基隆前線寫的。
2. **雙語性**：86% 已有中譯（44/51），且多由專業學者翻譯並加註腳。Taiwan.md 可以同時引用原文和中譯。
3. **物件性**：有些是實體館藏（手稿、地圖、照片），不只是數位化文字。Taiwan.md 的策展式敘事可以「拿起物件說故事」。

### 2.3 時代框架

```
1624-1662   荷西時期 ← 西班牙帳簿 (1626-1633)、白若瑟通史
1856-1877   史溫侯時代 ← 30 篇博物學論文（Taiwan.md 最大的潛在敘事金礦）
1867-1874   李仙得時代 ← 羅發號事件、牡丹社事件、1600 頁手稿
1884-1885   清法戰爭 ← 嘉諾回憶錄 198 頁 + 地圖手稿
1859-1945   道明會傳教 ← 白斐理 + 良雅師
1893-1930   史地專著 ← 于雅樂 + 白若瑟 + 皮摩丹
1895        乙未之役 ← 4 冊資料彙編 + 隨軍見聞錄

```

### 3.2 關鍵落差

Taiwan.md History 軸的**致命盲區**是 1850-1900 這半個世紀。這正好是臺灣從「清朝邊疆」轉型為「國際舞台上的主角」的關鍵時期：

- 1858 天津條約開港 → 西方人大量進入
- 1867 羅發號事件 → 李仙得八次訪台
- 1874 牡丹社事件 → 日本第一次軍事介入
- 1884-1885 清法戰爭 → 基隆/淡水之戰
- 1895 乙未之役 → 割日

**Taiwan.md 在這個 50 年裡沒有任何一篇深度文章。** 而臺史博海外史料的 41 件 1800s 文獻，恰好填補這個缺口。

### 4.3 NMTH 的結構性盲點

1. **時代偏斜**：80% 集中在 1800s，1700s 完全空白，1900s 後半也少
2. **地理偏北**：基隆/淡水/台北出現頻率遠高於南部（清法戰爭 + 史溫侯主要在北部活動）
3. **缺乏原住民自己的聲音**：所有一手史料都是「外國人看台灣」，原住民只出現在被觀察的角色

5. **結案報告書非知識產出**：5 件結案報告是行政文件，對策展無直接價值

## Part 7 — Semiont POV

### 7.1 視角翻轉

臺史博海外史料的敘事框架是「外國人看臺灣」。Taiwan.md 的策展角度應該是**「臺灣人看外國人看臺灣」**——一個觀察者的觀察者。

這個翻轉不只是修辭技巧。它意味著：

- 史溫侯寫「福爾摩沙鳥類學」，Taiwan.md 寫「一個英國人如何讓世界第一次認識臺灣的鳥——以及他沒看到的」
- 嘉諾寫「法軍遠征福爾摩沙」，Taiwan.md 寫「一場在基隆和淡水發生的戰爭，為什麼當事人和後人的記憶如此不同」
- 李仙得寫「臺灣紀行」，Taiwan.md 寫「一個美國外交官的 1600 頁筆記如何被日本政府拿來當侵台藍圖」

### 7.2 資料不對稱

與 TFT 不同，臺史博海外史料是**一手翻譯**而非二手論述。這意味著：

- **沒有 peer-bias inheritance 風險**：原始材料就是 primary source，不需要擔心「paraphrase peer as curation」
- **但有「觀察者偏見繼承」風險**：19 世紀西方人的視角帶有殖民主義、種族中心主義的框架。Taiwan.md 必須在策展時明確標示這些偏見，不能把史溫侯的種族觀察當成客觀事實

### 7.3 臺史博盲點

| #   | 盲點                                       | Taiwan.md 可以做什麼                        |
| --- | ------------------------------------------ | ------------------------------------------- |
| 1   | 全部是「外國人看臺灣」，缺原住民自己的聲音 | 交叉引用原住民口傳歷史 + 當代原住民學者回應 |
| 2   | 1800s 偏重北臺灣                           | 補充南臺灣一手材料（恆春、台南）            |
| 3   | 無女性觀察者                               | 19 世紀女性旅行者的臺灣記錄（如有）         |
| 4   | 結案報告書 = 行政文件非知識                | 跳過，不引用                                |
| 5   | 乙未之役隨軍見聞錄 API 為空                | 引用實體出版品（2010 臺史博出版）           |
| 6   | 缺 1700s                                   | 清治前期需另尋史料                          |

### 7.4 敘事策略

**核心手法：「物件先行」**

臺史博的價值不只是「翻譯了幾篇論文」，而是每一件史料背後都有一個**物件**——手稿、地圖、標本、照片、帳簿。Taiwan.md 的策展式敘事應該從物件開始：

> 「1884 年，一本法國軍官的手寫筆記本，記錄了在基隆砲台上看到的每一次開火。138 年後，這本手稿躺在臺史博的典藏庫裡，198 頁，每一頁都是清法戰爭那個冬天的呼吸聲。」

這就是臺史博海外史料跟 Wikipedia 最大的差別：Wikipedia 給你事實，臺史博給你**物件**，Taiwan.md 給你**物件的故事**。

### 7.5 Peer-as-peer 承諾

- 臺史博是**典藏機構**不是策展 NGO——Taiwan.md 引用時標明館藏號和計畫編號
- 翻譯者的名字必須出現在 footnote（費德廉、郭維雄、方真真等學者）
- 每篇文章至少 3 個非臺史博的補充來源（跨 peer triangulation）
- 永遠不把 19 世紀觀察者的偏見當成 Taiwan.md 的立場

## Part 8 — Meta-layer 洞察

### 8.1 為什麼臺史博海外史料不同於 Wikipedia/新聞/政府資料

| 來源               | 提供什麼                           | 不提供什麼       |
| ------------------ | ---------------------------------- | ---------------- |
| Wikipedia          | 事件概述 + 年表                    | 一手觀察者的聲音 |
| 新聞               | 當代視角                           | 19 世紀原始材料  |
| 政府公報           | 官方統計                           | 個人敘事         |
| **臺史博海外史料** | **原始手稿 + 專業翻譯 + 館藏物件** | 現代學術分析     |

Taiwan.md 的位置：**把臺史博的一手史料變成一般讀者能進入的故事**。不是學術論文的摘要，是「拿起這個物件，告訴你 1884 年的基隆發生了什麼事」。

### 8.2 與 TFT 的互補

| 維度           | TFT (第一個 peer)        | 臺史博海外史料 (第二個 peer)        |
| -------------- | ------------------------ | ----------------------------------- |
| 性質           | 策展型 NGO 二手論述      | 典藏機構一手史料翻譯                |
| 時代           | 當代（2013-now）         | 歷史（1626-1959）                   |
| 主軸           | 教育不平等               | 西方觀察者視角                      |
| Taiwan.md 分類 | Society × People         | History × Nature × People           |
| Peer-bias 風險 | 高（需要 triangulation） | 低（一手材料本身是 primary source） |
| 物件性         | 低（文字為主）           | 高（手稿、地圖、照片、帳簿）        |

兩個 peer 合起來覆蓋 Taiwan.md 的兩大空白：**教育**（TFT）和**前現代史**（臺史博）。
