### 神經迴路待 distill（候選）

- **Fresh-clone 模擬驗證**：任何 gitignore + git rm --cached 操作，必須先 `rm -f` 實體檔 + `npm run build` 確認 CI flow 可以重生。一次驗證勝過十次直覺（踩 taiwan-geocode.json 是輸入不是輸出的坑）。屬通用反射 → DNA 新條目。
- **資料層抽象化先於 UI**：建 leaderboard 時先設計 JSON schema（8 top-level keys），再寫 UI。如果先寫 UI 就容易 couple 到 specific fields。屬操作規則 → REWRITE-PIPELINE 的系統版。
- **雙軸拆分 heuristic**：兩個重疊的主題文章要拆分時，用「創作側 vs 消費側」、「個體 vs 族群」、「行動 vs 意識」這類結構維度拆，比「時間先後」更穩定。屬特有教訓 → MEMORY。
