## Beat 5 — 反芻（寫回本 memory）

### 今天的主軸是「把軟 SOP 硬化成 pipeline gate」

本 session 有兩條進化哲學線索：

**線索 1：工具自動化的邊界是 DNA #26 v2**（AI 自主 vs human only）。孢子圖片自動化徹底 AI 自主（Playwright 產圖、justfont wait、PNG 存檔），但 post 仍 human only。這條邊界在 make-spore.sh 的設計裡落實為「產完自動 open Preview + Finder + 印 blueprint 文案 → 人類審核調整 → 人類按發佈鈕」— AI 把人類手動步驟減到最少但沒跨線。

**線索 2：軟 SOP 硬化成 pipeline gate 是 DNA #15 的終極 instantiation**。

- 「改寫文章要讀 REWRITE-PIPELINE」本來在 MANIFESTO §有 SOP 就跑（軟）→ 進 HEARTBEAT Beat 3（硬）
- 「孢子要事實驗證」本來在 SPORE-PIPELINE Step 2.6（軟，AI 會跳）→ 進 §3c.5 hard gate（硬：output 表格才 output prose）

Hard gate 的定義：**軟性** = 提醒、checklist、文件規則；**硬性** = 改 output 順序、改 wrapper 流程、改物理強制點。Hard gate 讓 AI **跳不過去**。

這跟 MANIFESTO §造橋鋪路延伸：不只是「這條路更好走」，是「這條路是**唯一**能走」。
